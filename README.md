# MyTix

MyTix adalah aplikasi mobile berbasis Android yang dikembangkan sebagai proyek UAS Mobile Programming. Aplikasi ini memudahkan pengguna untuk melihat informasi event kampus, melakukan pemesanan tiket, memilih metode pembayaran, serta melihat riwayat tiket yang telah dipesan.

## Fitur

- Login & Register
- Home menampilkan daftar event
- Detail Event
- Pemesanan Tiket
- Pilihan Metode Pembayaran
  - QRIS
  - DANA
  - ShopeePay
  - BCA Virtual Account
- Riwayat Tiket
  - Semua
  - Akan Datang
  - Selesai
  - Dibatalkan
- Profile Pengguna
- Pencarian Event
